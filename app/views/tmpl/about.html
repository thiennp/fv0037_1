This demo uses Angular.js. Each draggable widget can have it's own controller and scope with it's own data calls and logic. Not only does this demo show off a neat visual interaction, but it boasts a functional, thought out solution to the seperation of concerns for each widget.

Feel free to review my code and suggest a better way to do this. I tried to get it to work with UI-router, but there didn't seem to be an effective way that didn't sacrifice flexibility.

Other stuff used...

  • Sortable.js
  • UI-Router
  • FontAwesome
  • Underscore.js
